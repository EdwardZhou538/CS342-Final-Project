The dataset used was the Kaggle Alzheimer's Dataset which can be found on this page: https://www.kaggle.com/datasets/tourist55/alzheimers-dataset-4-class-of-images.
Data consists of 6400 MRI scans of different individuals' brains, each classified on 4 different levels of dementia: No Dementia, Very Mild Dementia, Mild Dementia,
and Moderate Dementia. Image size is 208x176 for all images.
